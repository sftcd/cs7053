CS7NS5/CSU44032 Security and Privacy

This is the 2020 version. Last updated 20200218 by stephen.farrell@cs.tcd.ie

  • The canonical URL for this stuff is: https://down.dsg.cs.tcd.ie/cs7053/ and the latest source is in a github repo at https://github.com/sftcd/cs7053
  • Over the years, this course has had lots of course codes, it was also previously CS4407, CS7453 and CS7074, and before that CS7012 and before that NDS106 (the last two links are to the old lectures etc.) But when I created the github repo it was CS7053, so I stick with that for the URLs:-) The content overlaps a good bit but always evolves.
  • The Lecture slides section below has links to ppt/pdf versions of all the lectures for the course. The PDFs should be fine as there're no fancy PPT features used.
  • The Materials section below has links to, and/or local copies of, some background information/papers etc. For exam purposes, you won't need anything else other than easily available things like RFCs and the materials below that are mentioned in class (You are of course encouraged to read more widely).
  • You can look at loads of old exams with both questions and answers/marking schemes.
  • There will be two assignments, totalling 20% of the marks for the module (so the exam is worth 80%).
  • Note that I will be changing the slides as I go. Depending on stuff that's more than two weeks ahead of time may be wasted effort:-)
